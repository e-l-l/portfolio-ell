# Portfolio | ell

A clean, responsive portfolio site built with Next.js.

## Tech Stack

- Next.js
- Tailwind CSS
- shadcn/ui
- Three.js

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
portfolio-ell/
├── app/        # Pages
├── components/ # UI components
├── public/     # Static assets
└── styles/     # CSS
```
